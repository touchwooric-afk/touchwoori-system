# TOUCHWOORI 영수증 관리 시스템 PRD

> 마지막 업데이트: 2026-03-27

---

## 1. 서비스 개요

### 목적
교회 고등부의 재정 집행 투명성 확보. 교사들이 지출한 영수증을 디지털로 제출하고, 회계교사가 검토·승인하여 장부에 자동 연동. 결산 시 감사 대비 지출증빙 PDF를 생성한다.

### 사용자 역할

| 역할 | 설명 |
|------|------|
| `master` | 총무/담당자. 시스템 전체 관리 |
| `sub_master` | 교육목사 등. 사용자 관리·역할 부여만 가능. 재정 기능 없음 |
| `accountant` | 회계교사. 영수증 승인, 장부 관리, 결산, 카테고리 관리 |
| `auditor` | 교육위원장 등. 전체 부서 열람 전용 (쓰기 불가) |
| `teacher` | 일반교사. 영수증 제출·조회, 장부 조회 |

## 2. 기능 명세 (현재 구현 상태)

### 2.1 인증
- 이메일/비밀번호 로그인
- 회원가입 시 `pending` 상태 → master/sub_master 승인 필요
- 비밀번호 재설정 (이메일 링크)
- 역할별 라우트 가드 (`middleware.ts`)

### 2.2 사용자 관리 (`/master/users`)
- 가입 신청자 승인/거절, 역할 변경, 계정 비활성화
- master, sub_master 모두 접근 가능
- sub_master는 master 계정 수정 불가

### 2.3 카테고리 관리 (`/master/categories`)
- 수입/지출 카테고리 CRUD
- 키워드 설정 (OCR/엑셀 자동 매칭용), 색상 지정
- **master + accountant** 모두 편집 가능 (부서 구분 없이 전체 공통)

### 2.4 직분 관리 (`/master/positions`)
- 직분 CRUD (목사, 전도사, 장로, 권사, 집사 등)
- master 전용

### 2.5 영수증 제출

**단건 제출** (`/receipts/submit`) — 모바일 최적화
- 사진 업로드, OCR 자동 파싱 (날짜·금액·업체명)
- 이미지 압축: 업로드 전 1200px/80% 자동 압축
- 장부 항목 후보 자동 매칭 (금액 100pt + 항목명 유사도 50pt + 날짜근접 최대 10pt)
- 연동 또는 미연동 선택 후 제출
- 중복 감지: 같은 부서+날짜+금액 하드블록 (연동 모드 제외)
- 자주 쓰는 계좌 즐겨찾기 (account_favorites)

**일괄 업로드** (`/receipts/upload`)
- 다중 파일 + OCR 자동 파싱
- 유효하지 않은 항목은 error 상태 보류 → 수정 후 재저장

### 2.6 내 제출 내역 (`/receipts/my`)
- 상태별 필터 (전체/대기중/승인됨/반려됨)
- 항목일 + 제출일 표시
- accountant/master는 제출자 이름 함께 표시
- 반려 시 재제출 가능
- 이미지 hover 미리보기

### 2.7 영수증 승인 (`/receipts/pending`)
- 미승인 목록 + 일괄 승인
- 개별 승인: 새 항목 추가 / 기존 항목 연동 / 이미 연동됨 자동 감지
- 반려 (사유 입력)
- 승인 시 금액 수정 가능 (approved_amount)

### 2.8 회계장부 (`/ledger`)
- 다중 장부 지원 (본 장부 + 특수 장부)
- 항목별 누적 잔액 표시
- 필터: 기간, 카테고리 클릭 태그, 검색어 (텍스트 + 금액 숫자 검색), 영수증 연동 여부
- 이미지 hover 미리보기 (우측 fixed 팝업)
- 영수증 연동 항목 수정 시 경고 배너 표시
- 다중 항목 일괄 입력 모달
- 엑셀 내보내기 (현재 필터 적용)

### 2.9 결산 및 지출증빙 (`/settlements`)
- 기간 선택: 직접 입력 / 전반기(12~4월) / 후반기(5~11월)
- 미리보기: 수입부(초록) + 지출부(빨강) 결산표 + 기말 잔액
- PDF 생성:
  - 1페이지: 수입·지출 카테고리 합계표 + 남는 공간에 영수증 자동 채움
  - 2페이지~: 영수증 2×2 그리드 (EXIF 자동 회전, 최대 1200px 압축)
- 브라우저 미리보기 후 PDF 다운로드

### 2.10 엑셀 연동
- **내보내기** (`/excel/export`): 장부 + 기간 선택 → xlsx 다운로드
- **가져오기** (`/excel`): xlsx 업로드 → 행별 카테고리 매칭 미리보기 → 장부 임포트

### 2.11 대시보드 (`/`)
- 역할별 통계 카드 및 바로가기
- 월별 수입/지출 추이 차트, 카테고리별 지출 파이 차트 (teacher 제외)
- 부서 로고 표시 (Supabase Storage `department-banners` 버킷, 인사말 우측)

## 3. 화면 구조

### 사이드바 메뉴

| 그룹 | 메뉴 | 권한 |
|------|------|------|
| 홈 | 대시보드 | 전체 |
| 시스템 관리 | 사용자 관리 | master, sub_master |
| 시스템 관리 | 카테고리 관리, 직분 관리 | master 전용 |
| 영수증 | 영수증 제출, 내 제출 내역 | teacher, accountant, master |
| 영수증 | 미승인 영수증, 직접 입력 | accountant, master |
| 회계장부 | 회계장부 조회 | 전체 |
| 회계장부 | 장부 관리, 엑셀 내보내기 | accountant, master |
| 결산 | 결산 및 지출증빙 | 전체 |

> auditor, sub_master는 영수증 제출·승인 메뉴 없음

### 모바일 하단 탭
- teacher: 홈 / 제출 / 내역 / 장부
- accountant: 홈 / 미승인 / 장부 / 결산
- master: 홈 / 관리 / 장부 / 결산

## 5. 변경 이력

### 2026-03-27 — 현재 버전

**제거**
- 결산기 관리 기능 전체 제거 (`/master/settlements`, `/settlements/history`, `/api/settlements`, `Settlement` 타입)
- 감사 내역 아카이브 기능 제거

**권한 확장**
- `accountant`도 카테고리 관리 가능 (카테고리는 전체 공통, 부서 구분 없음)
- `sub_master` 장부 항목 편집 권한 추가 (isEditor에 포함)

**영수증**
- 내 제출 내역에 제출일 표시 추가
- accountant/master의 내 제출 내역에 제출자 이름 표시
- 전체 영수증 관련 화면에서 제출자 표시 일관화

**장부**
- 카테고리 태그 클릭으로 필터링/해제 기능
- 금액 숫자 입력 시 income/expense 컬럼도 검색
- 영수증 연동 항목 수정 시 경고 배너
- 이미지 hover 미리보기 (fixed 우측 팝업)

**결산 PDF**
- 이미지 최대 1200px / quality 0.75 압축 (15MB+ 문제 해결)
- 수입부/지출부 테이블 색상 일관화 (수입: 초록, 지출: 빨강)
- 탭 이름 변경: "결산 PDF" → "결산 및 지출증빙"

**레이아웃**
- 사이드바 너비 확대: 224px → 256px
- 대시보드 부서 로고 추가 (Supabase Storage, 인사말 우측 배치)
