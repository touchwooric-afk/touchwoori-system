# 고등부 영수증 관리 시스템 -- 디자인 문서

> 생성일: 2026-03-19

## 한 줄 요약

교사가 스마트폰으로 영수증 제출 → 회계 교사가 승인 → 장부 자동 반영.
마스터(시스템 관리자 + 담당 목사)가 전체 시스템을 관리하고, 결산 PDF를 A4 최소 장수로 출력.

## 역할 구조

```
마스터 (master) ─── 최고 권한, 모든 부서 CRUD, 시스템 관리
  ├── 회계 교사 (accountant) ─── 자기 부서 장부 CRUD, 영수증 승인, PDF·엑셀
  └── 교사 (teacher) ─── 자기 부서 열람(읽기 전용), 영수증 제출

[Phase 3 추가]
  └── 감사 (auditor) ─── 전체 부서 열람만 (교육위원장/장로)

* 직분(담당교역자, 부장교사 등)은 표시용. 시스템 역할(master/accountant/teacher)이 권한을 결정.
```

---

## 핵심 흐름

```
교사 (스마트폰)              회계 교사 (PC)
  사진 촬영                    미승인 목록 확인
  → OCR 자동 인식              → 검토·수정
  → 제출                       → 승인
                               → 장부항목 자동 생성 ✅
                               → 장부 직접 편집도 가능

  내 제출 내역에서              결산 PDF 출력 (지출만)
  상태 확인 가능                → A4 최소 장수로 인쇄
  장부 열람 가능 (읽기 전용)
```
